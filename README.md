# GSoC 2026 — Lumen + Xarray Integration

Biweekly blog by [Aman Kumar](https://github.com/ghostiee-11) for
Google Summer of Code 2026 with [HoloViz](https://holoviz.org) under
the [NumFOCUS](https://numfocus.org) umbrella.

**Live site:** <https://ghostiee-11.github.io/gsoc26/>

Built with [Jekyll](https://jekyllrb.com/) and the `minima` theme on
GitHub Pages.

## Adding a new post

Create `_posts/YYYY-MM-DD-slug.md` with the standard Jekyll front
matter, commit, push to `main`. The site rebuilds automatically.

## Local preview (optional)

```sh
bundle exec jekyll serve
```
